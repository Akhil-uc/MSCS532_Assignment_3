# How to Run

## Requirements

- Python 3.x installed on your system.

## Run Randomized Quicksort

```bash
python randomized_quicksort.py
```

## Run Deterministic Quicksort

```bash
python deterministic_quicksort.py
```

## Run Hash Table with Chaining

```bash
python hash_table_chaining.py
```

## Run Performance Comparison

```bash
python performance_test.py
```

The performance test compares **Randomized Quicksort** and **Deterministic Quicksort** using:

- Random arrays
- Sorted arrays
- Reverse-sorted arrays
- Arrays with duplicate elements

for array sizes of **1000**, **5000**, and **10000**.
